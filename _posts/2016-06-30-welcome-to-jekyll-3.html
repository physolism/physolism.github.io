สวัสดีจ้า ! สำหรับบล็อคนี้ก็เป็นบล็อคแรก ก็ไม่รู้จะพูดถึงอะไรดี เอาเป็นว่าพูดถึงที่มาที่ไปของ blog นี้ละกัน

นี้เป็นครั้งแรกที่เริ่มเรียนรู้การทำเว็บ การใช้ html, css, js ในการเขียนบล็อคเลย ซึ่งไอเจ้า Jekyll นี้ที่จริงมันก็ไม่ยากนะ แต่ด้วยความรู้การทำเว็บระดับเด็กอนุบาลมากจึงทำให้ใช้เวลาในการสร้าง blog นี้แบบเลือดตาแทบจะไหล แต่สุดท้ายก็ออกมาเป็นรูปเป็นร่างจนได้ภายในเวลา 3 วัน (ที่จริงเสียเวลาส่วนใหญ่ไปกับการ deploy ขึ้น github - -") สำหรับเจ้า Jekyll เหมาะสำหรับมือใหม่และไม่ใหม่ที่สนใจจะมี blog ที่เป็น static web ของตัวเอง เพราะมันง่ายและสะดวกมากเลยทีเดียว

สุดท้ายนี้ ฝาก blog เล็กๆนี้ด้วยละ !
